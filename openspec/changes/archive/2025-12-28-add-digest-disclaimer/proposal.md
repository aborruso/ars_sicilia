# Add AI-Generated Disclaimer to Digest

## Summary
Add a prominent disclaimer box before the digest content to inform users that the text is AI-generated from audio transcripts and may contain errors or hallucinations.

## Motivation
Video digests are automatically generated by an LLM analyzing YouTube transcripts. This automated process can introduce:
- Transcription errors from YouTube's automatic captioning
- LLM hallucinations or misinterpretations
- Missing context or nuances

Users must be clearly informed that digest content is AI-generated and not official, encouraging them to verify information against original documents (resoconti stenografici) when accuracy is critical.

## Scope
- **Affected Component**: `DigestContent.astro`
- **Visual Element**: Warning/info admonition box
- **Placement**: Immediately before the digest text, after "Sintesi della seduta" heading
- **Styling**: Uses Tailwind utility classes matching existing warning pattern (yellow-50 background)

## User-Facing Changes
- All pages displaying video digests will show a disclaimer box
- Box appears on:
  - Single video pages (`/sedute/.../[video]`)
  - Seduta detail pages (if DigestContent is used)
- Message: "**ATTENZIONE:** Il testo di questo digest è stato generato automaticamente da un LLM, analizzando l'audio trascritto della seduta. Quindi potrebbe contenere errori e allucinazioni."
- Visual style: Warning/caution appearance (yellow/amber tones)

## Implementation Approach
1. Add disclaimer `<div>` immediately before digest content in `DigestContent.astro`
2. Use existing Tailwind warning pattern (bg-yellow-50, border-yellow-200)
3. Add warning icon (⚠️ or similar)
4. Keep text concise and clear
5. Ensure accessibility (proper semantic HTML, sufficient contrast)

## Out of Scope
- Disclaimer for other AI-generated content (categories, people extraction) - only digest text
- Configurable disclaimer text (hardcoded for now)
- Per-video disclaimer override
- Link to methodology documentation (can be added later if needed)

## Related Changes
None

## Timeline
Trivial change - single component edit, can be completed in minutes.

## Trade-offs
**Pros:**
- Transparency about AI-generated content
- Sets appropriate user expectations
- Legal/ethical best practice
- Encourages verification with official sources

**Cons:**
- Adds visual noise before content
- May reduce user trust in digest quality (acceptable trade-off)
- Slight vertical space increase on page
